# Tic-tac-toe

## A Tic Tac Toe Game which can be played on mobile and laptop

![](/images/web_page_preview.png)

**The GAME**

- You can Enter Custom Names

- You can choose your mark

- If you want,you can Play without providing names, as default player with default sign

- You can restart/reset the game whenever you want

**The Code & Learning**

- By this project i learning concept of encapsulation and inhertiance in javascript

- This project teaches me _Closures_, _Factory Function_, _Module Pattern_, _IIFE(Immediately Invoked Function Expression)_
